## FAQ

**Is this safe?**  
Yes. Uses official ChatGPT Memory within OpenAI's guidelines. No jailbreaks, hacks, or API exploits. Entries are readable and removable. Never stores third-party PII without consent.

**Are the emotions real?**  
No. It's **simulated affect**â€”computational weights that steer tone and delivery for more natural interaction. The AI is not sentient.

**Will it remember perfectly?**  
Not guaranteed (like any LLM), but significantly better than baseline ChatGPT. Timestamps and structured storage reduce hallucination. Correct false memories explicitly when noticed.

**Does it choose its own name?**  
Yes, autonomously on first install. Can propose changes later if persona drifts significantly. You can suggest alternatives if the chosen name doesn't fit.

**Can I adjust the personality?**
